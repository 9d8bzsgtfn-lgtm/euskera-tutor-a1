# Gramatika Erreferentzia - Euskera A1

## 1. Sistema de Casos (Kasu Sistema)

El euskera tiene un sistema de casos que marca la funcion de las palabras en la oracion.

### 1.1 Caso Absolutivo (NOR)

**Funcion:** Sujeto de verbos intransitivos, objeto de verbos transitivos
**Marca singular:** -a (determinado), -0 (indeterminado)
**Marca plural:** -ak (determinado)

| Forma | Ejemplo | Traduccion |
|-------|---------|------------|
| Singular det. | Umea etorri da | El nino ha venido |
| Singular indet. | Ume bat etorri da | Un nino ha venido |
| Plural det. | Umeak etorri dira | Los ninos han venido |

### 1.2 Caso Ergativo (NORK)

**Funcion:** Sujeto de verbos transitivos
**Marca singular:** -k (despues de vocal), -ek (despues de consonante)
**Marca plural:** -ek

| Forma | Ejemplo | Traduccion |
|-------|---------|------------|
| Singular | Umeak ogia jan du | El nino ha comido pan |
| Plural | Umeek ogia jan dute | Los ninos han comido pan |

**Nota importante:** El ergativo solo se usa con verbos transitivos (que tienen objeto directo).

### 1.3 Caso Dativo (NORI)

**Funcion:** Objeto indirecto
**Marca singular:** -(r)i
**Marca plural:** -ei

| Forma | Ejemplo | Traduccion |
|-------|---------|------------|
| Singular | Amari eman diot | Se lo he dado a mama |
| Plural | Umeei eman diet | Se lo he dado a los ninos |

### 1.4 Caso Inesivo (NON)

**Funcion:** Ubicacion estatica (donde)
**Marca:** -n, -an, -ean (segun terminacion)

| Base | + Inesivo | Traduccion |
|------|-----------|------------|
| etxe (casa) | etxean | en casa |
| eskola (escuela) | eskolan | en la escuela |
| Bilbo | Bilbon | en Bilbao |
| Madrid | Madrilen | en Madrid |

### 1.5 Caso Adlativo (NORA)

**Funcion:** Direccion hacia (a donde)
**Marca:** -ra, -eta (plural)

| Base | + Adlativo | Traduccion |
|------|------------|------------|
| etxe | etxera | a casa |
| eskola | eskolara | a la escuela |
| Bilbo | Bilbora | a Bilbao |

### 1.6 Caso Ablativo (NONDIK)

**Funcion:** Procedencia (de donde)
**Marca:** -tik, -etik

| Base | + Ablativo | Traduccion |
|------|------------|------------|
| etxe | etxetik | de casa |
| eskola | eskolatik | de la escuela |
| Bilbo | Bilbotik | de Bilbao |

## 2. Verbos Auxiliares (Aditz Laguntzaileak)

### 2.1 IZAN - Ser/Estar (verbos intransitivos)

**Presente (Orainaldia)**

| Persona | Singular | Plural |
|---------|----------|--------|
| Ni | naiz | - |
| Zu | zara | - |
| Bera | da | - |
| Gu | - | gara |
| Zuek | - | zarete |
| Haiek | - | dira |

**Ejemplos:**
- Ni ikaslea naiz (Yo soy estudiante)
- Zu handia zara (Tu eres grande)
- Bera medikua da (El/Ella es medico)
- Gu lagunak gara (Nosotros somos amigos)
- Zuek Bilbokoak zarete (Vosotros sois de Bilbao)
- Haiek hemen dira (Ellos estan aqui)

**Pasado (Iragana)**

| Persona | Singular | Plural |
|---------|----------|--------|
| Ni | nintzen | - |
| Zu | zinen | - |
| Bera | zen | - |
| Gu | - | ginen |
| Zuek | - | zineten |
| Haiek | - | ziren |

### 2.2 UKAN - Haber/Tener (verbos transitivos)

**Presente NOR-NORK (Objeto 3a persona singular)**

| NORK / Sujeto | du (lo/la) | dute (ellos lo) |
|---------------|------------|-----------------|
| Nik | dut | - |
| Zuk | duzu | - |
| Berak | du | - |
| Guk | dugu | - |
| Zuek | duzue | - |
| Haiek | - | dute |

**Ejemplos:**
- Nik liburua irakurtzen dut (Yo leo el libro)

## 3. Estructura de la Oracion

### 3.1 Orden SOV

El euskera sigue el orden Sujeto-Objeto-Verbo:

| Castellano (SVO) | Euskera (SOV) |
|------------------|---------------|
| Yo como pan | Nik ogia jaten dut |
| Maria lee un libro | Mariak liburu bat irakurtzen du |

### 3.2 Posicion del Verbo

El verbo conjugado va **al final** de la oracion:
- Ni etxean **nago** (Yo estoy en casa)
- Nik kafea **nahi dut** (Yo quiero cafe)

### 3.3 Preguntas

**Preguntas Si/No:**
Anadir particula **al** o entonacion ascendente:
- Zu ikaslea **al** zara? (Eres estudiante?)
- Etxean zaude? (Estas en casa?)

**Preguntas con interrogativo:**
| Interrogativo | Significado | Ejemplo |
|---------------|-------------|---------|
| Nor? | Quien? | Nor da? (Quien es?) |
| Zer? | Que? | Zer da hau? (Que es esto?) |
| Non? | Donde? | Non zaude? (Donde estas?) |
| Nora? | A donde? | Nora zoaz? (A donde vas?) |
| Nondik? | De donde? | Nondik zatoz? (De donde vienes?) |
| Noiz? | Cuando? | Noiz etorriko zara? (Cuando vendras?) |
| Nola? | Como? | Nola zaude? (Como estas?) |
| Zergatik? | Por que? | Zergatik? (Por que?) |
| Zenbat? | Cuanto/s? | Zenbat? (Cuanto?) |

## 4. Articulo y Determinantes

### 4.1 Articulo Definido (sufijo)

| Tipo | Marca | Ejemplo |
|------|-------|---------|
| Singular | -a | etxea (la casa) |
| Plural | -ak | etxeak (las casas) |

**Nota:** El articulo se anade al final de la palabra, no antes.

### 4.2 Articulo Indefinido

| Tipo | Forma | Ejemplo |
|------|-------|---------|
| Singular | bat | etxe bat (una casa) |
| Plural | batzuk | etxe batzuk (unas casas) |

### 4.3 Demostrativos

| Distancia | Singular | Plural |
|-----------|----------|--------|
| Cerca | hau (este/a) | hauek (estos/as) |
| Media | hori (ese/a) | horiek (esos/as) |
| Lejos | hura (aquel/la) | haiek (aquellos/as) |

## 7. Adjetivos

### 7.1 Posicion

Los adjetivos van **despues** del sustantivo:
- etxe handia (la casa grande)
- gizon altua (el hombre alto)
- No: *handi etxea

### 7.2 Concordancia

El adjetivo toma el articulo, no el sustantivo:
- etxe handi**a** (la casa grande)
- etxe handi**ak** (las casas grandes)

## 9. Negacion

### 9.1 Estructura basica

La negacion se forma con **ez** antes del verbo:
- Ni ez naiz ikaslea (Yo no soy estudiante)
- Nik ez dut ulertzen (Yo no entiendo)

### 9.2 Doble negacion

Con palabras negativas, se mantiene **ez**:
- Ez dut **ezer** ikusten (No veo nada)
- Ez dago **inor** (No hay nadie)

| Afirmativo | Negativo |
|------------|----------|
| zerbait (algo) | ezer (nada) |
| norbait (alguien) | inor (nadie) |
| nonbait (en algun lugar) | inon (en ningun lugar) |
