Praktika: Janaria eta Edaria

ARGIBIDEAK

Aukeratu erantzun zuzena ariketa bakoitzean.

Selecciona la respuesta correcta en cada ejercicio.

1. Ariketa: NAHI IZAN aditza (Verbo QUERER)

______ kafea nahi dut. (Yo quiero cafe.)

Ni
Nik
Niri
Nire

Oso ondo! Con NAHI IZAN usamos el ergativo: "Nik"

Ez zuzena. NAHI IZAN requiere caso NORK (ergativo): "Nik", no "Ni".

Zuk ura nahi ______. (Tu quieres agua.)

dut
duzu
du
zaizu

Bikain! "Zuk" + "nahi duzu" = Tu quieres

Ez zuzena. Con "Zuk", el auxiliar es "duzu".

Guk garagardoa nahi ______. (Nosotros queremos cerveza.)

dut
dute
dugu
duzue

Primeran! "Guk" + "nahi dugu" = Nosotros queremos

Ez zuzena. Con "Guk" (nosotros), el auxiliar es "dugu".

Haiek ogia nahi ______. (Ellos quieren pan.)

du
dugu
duzue
dute

Oso ondo! "Haiek" + "nahi dute" = Ellos quieren

Ez zuzena. Con "Haiek" (ellos), el auxiliar es "dute".

2. Ariketa: GUSTATU aditza (Verbo GUSTAR)

______ kafea gustatzen zait. (A mi me gusta el cafe.)

Ni
Nik
Niri
Nire

Bikain! Con GUSTATU usamos el dativo: "Niri"

Ez zuzena. GUSTATU requiere caso NORI (dativo): "Niri" = a mi.

Zuri arraina gustatzen ______. (A ti te gusta el pescado.)

zait
zaizu
zaio
duzu

Primeran! "Zuri" + "gustatzen zaizu" = A ti te gusta

Ez zuzena. Con "Zuri", el auxiliar de GUSTATU es "zaizu".

Hari pintxoak gustatzen ______. (A el/ella le gustan los pintxos.)

zaio
zaizkio
zaizkie
du

Oso ondo! "Pintxoak" es plural, entonces "zaizkio" (le gustan)

Ez zuzena. Como "pintxoak" es plural, usamos "zaizkio" (le gustan).

Guri sagardoa gustatzen ______. (A nosotros nos gusta la sidra.)

zait
zaizu
zaigu
dugu

Bikain! "Guri" + "gustatzen zaigu" = A nosotros nos gusta

Ez zuzena. Con "Guri" (a nosotros), el auxiliar es "zaigu".

3. Ariketa: NORI kasua (Caso dativo)

Cual es el dativo de "zu" (tu)?

zuk
zuri
zure
zuek

Oso ondo! "Zuri" = a ti

Ez zuzena. El dativo de "zu" es "zuri" (a ti).

Cual es el dativo de "haiek" (ellos)?

hari
haien
haiei
haiek

Bikain! "Haiei" = a ellos/ellas

Ez zuzena. El dativo de "haiek" es "haiei" (a ellos).

4. Ariketa: Hiztegia (Vocabulario de comida)

Como se dice "la cerveza" en euskera?

ardoa
garagardoa
sagardoa
txakolina

Oso ondo! "Garagardoa" = la cerveza

Ez zuzena. Cerveza = garagardoa, vino = ardoa, sidra = sagardoa.

Que es un "pintxo"?

Una bebida tipica vasca
Una pequena tapa sobre pan
Un tipo de queso
Un postre tradicional

Bikain! Los pintxos son pequenas tapas servidas sobre pan con un palillo.

Ez zuzena. Un pintxo es una pequena tapa tipica del Pais Vasco.

Que significa "Gose naiz"?

Tengo sed
Tengo hambre
Estoy lleno
Estoy cansado

Oso ondo! "Gose naiz" = Tengo hambre

Ez zuzena. "Gose naiz" = Tengo hambre. "Egarri naiz" = Tengo sed.

Como pides la cuenta en un restaurante?

Kontua, mesedez.
Kafea, mesedez.
Ogia, mesedez.
Ura, mesedez.

Bikain! "Kontua, mesedez" = La cuenta, por favor.

Ez zuzena. "Kontua" = la cuenta.

5. Ariketa: NAHI DUT vs GUSTATZEN ZAIT

Cual es la forma correcta para "Quiero cafe"?

Nik kafea nahi dut.
Niri kafea nahi dut.
Nik kafea gustatzen zait.
Ni kafea nahi dut.

Primeran! NAHI IZAN usa caso NORK: "Nik kafea nahi dut."

Ez zuzena. Para "querer" usamos NORK (ergativo): "Nik".

Cual es la forma correcta para "Me gusta el cafe"?

Nik kafea gustatzen dut.
Ni kafea gustatzen zait.
Niri kafea gustatzen zait.
Niri kafea nahi zait.

Bikain! GUSTATU usa caso NORI: "Niri kafea gustatzen zait."

Ez zuzena. Para "gustar" usamos NORI (dativo): "Niri" y auxiliar "zait".

PRAKTIKA GEHIAGO

Ariketa hauek egin ondoren, saiatu txatean tutorearekin:

  • Eskatu janaria: "Garagardo bat nahi dut, mesedez"
  • Adierazi gustuak: "Niri arraina gustatzen zait"
  • Galdetu tutoreari: "Zer nahi duzu jan?"

Despues de estos ejercicios, intenta en el chat con el tutor:

  • Pide comida: "Quiero una cerveza, por favor"
  • Expresa gustos: "Me gusta el pescado"
  • Pregunta al tutor: "Que quieres comer?"
