Praktika: Osasuna eta Gorputza

ARGIBIDEAK

Aukeratu erantzun zuzena ariketa bakoitzean.

Selecciona la respuesta correcta en cada ejercicio.

1. Ariketa: Sentsazioak UKAN-ekin (Sensaciones con UKAN)

Como dices "Tengo hambre" en euskera?

Gose dut
Gosea daukat
Gose naiz
Gosea dut

Oso ondo! Las sensaciones usan UKAN sin articulo: "Gose dut"

Ez zuzena. Las sensaciones usan UKAN: Gose + dut (sin articulo)

______ duzu? (Tienes frio?)

Hotza
Hotz
Hotzik
Hotzeko

Bikain! "Hotz duzu?" - la sensacion va sin articulo.

Ez zuzena. Para sensaciones: Hotz + duzu (sin articulo -a)

Berak lo ______. (El/Ella tiene sueno.)

dut
duzu
du
dugu

Primeran! "Berak" (el/ella) + "du"

Ez zuzena. Con "berak" el verbo es "du".

Guk egarri ______. (Nosotros tenemos sed.)

dut
dute
duzue
dugu

Oso ondo! "Guk" (nosotros) + "dugu"

Ez zuzena. Con "guk" el verbo es "dugu".

2. Ariketa: EDUKI aditza (Verbo EDUKI - posesion)

Nik autoa ______. (Yo tengo un coche.)

dut
daukat
naiz
dago

Bikain! Para posesion fisica usamos EDUKI: "daukat"

Ez zuzena. Para posesion se usa EDUKI: daukat, daukazu, dauka...

Zuk txakurra ______? (Tu tienes un perro?)

daukat
daukazu
dauka
daukagu

Primeran! "Zuk" (tu) + "daukazu"

Ez zuzena. Con "zuk" el verbo EDUKI es "daukazu".

Cual es la diferencia correcta entre UKAN y EDUKI?

Son sinonimos, se pueden usar indistintamente
EDUKI para sensaciones, UKAN para posesion
UKAN para sensaciones, EDUKI para posesion
UKAN solo en pasado, EDUKI solo en presente

Oso ondo! UKAN = sensaciones (gose dut), EDUKI = posesion (autoa daukat)

Ez zuzena. UKAN se usa para sensaciones, EDUKI para posesion fisica.

3. Ariketa: Agintera (Imperativo)

Como dices "Ven!" en euskera (forma formal)?

Joan zaitez!
Etor zaitez!
Hartu ezazu!
Egin ezazu!

Bikain! "Etor zaitez!" = Ven! (formal)

Ez zuzena. "Etorri" = venir, imperativo formal: "Etor zaitez!"

"Hartu ezazu!" significa:

Haz!
Ve!
Coge!/Toma!
Ven!

Primeran! "Hartu" = coger/tomar

Ez zuzena. "Hartu" significa coger o tomar.

"Ura ______ !" (Bebe agua!)

jan
edan
hartu
eman

Oso ondo! "Edan" = beber

Ez zuzena. "Edan" = beber, "Jan" = comer

4. Ariketa: Gorputz atalak (Partes del cuerpo)

Que parte del cuerpo es "burua"?

El brazo
La mano
La cabeza
La pierna

Bikain! "Burua" = la cabeza

Ez zuzena. "Burua" es la cabeza.

Como se dice "los ojos" en euskera?

Belarriak
Begiak
Hortzak
Hatzak

Primeran! "Begiak" = los ojos

Ez zuzena. Begiak = ojos, Belarriak = orejas, Hortzak = dientes, Hatzak = dedos

"Sabeleko mina dut" significa:

Tengo dolor de cabeza
Tengo dolor de espalda
Tengo dolor de estomago
Tengo dolor de garganta

Oso ondo! "Sabela" = estomago, "Sabeleko mina" = dolor de estomago

Ez zuzena. "Sabela" es el estomago.

Quien es el "sendagilea"?

El enfermero
El medico
El paciente
El farmaceutico

Bikain! "Sendagilea" = el medico

Ez zuzena. "Sendagilea" es el medico, "Erizaina" es el enfermero.

PRAKTIKA GEHIAGO

Ariketa hauek egin ondoren, saiatu txatean tutorearekin:

  • Esan nola zauden: "Gaixo nago" edo "Ondo nago"
  • Esan zer mina daukazun: "Buruko mina dut"
  • Erabili aginterak: "Ura edan!" "Atseden hartu!"

Despues de estos ejercicios, intenta en el chat con el tutor:

  • Di como estas: "Gaixo nago" o "Ondo nago"
  • Di que dolor tienes: "Buruko mina dut"
  • Usa imperativos: "Ura edan!" "Atseden hartu!"
