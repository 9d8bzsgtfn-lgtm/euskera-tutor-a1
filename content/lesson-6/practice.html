Praktika: Hiria eta Garraioa

ARGIBIDEAK

Aukeratu erantzun zuzena ariketa bakoitzean.

Selecciona la respuesta correcta en cada ejercicio.

1. Ariketa: NON / NORA / NONDIK

Ni etxe______ nago. (Estoy en casa.)

-an
-ra
-tik
-ko

Oso ondo! Para "estar en" usamos NON: etxe + an = etxean

Ez zuzena. "Estar en" requiere caso NON (-an): etxean.

Ni Bilbo______ noa. (Voy a Bilbao.)

-n
-ra
-tik
-an

Bikain! Para "ir hacia" usamos NORA: Bilbo + ra = Bilbora

Ez zuzena. "Ir hacia" requiere caso NORA (-ra): Bilbora.

Ni Donostia______ nator. (Vengo de Donostia.)

-ra
-n
-tik
-an

Primeran! Para "venir de" usamos NONDIK: Donostia + tik = Donostiatik

Ez zuzena. "Venir de" requiere caso NONDIK (-tik): Donostiatik.

Geltoki______ trena hartzen dut. (En la estacion cojo el tren.)

-an
-ra
-tik
-ko

Oso ondo! "En la estacion" = geltokian (NON)

Ez zuzena. "En la estacion" usa caso NON: geltokian.

2. Ariketa: JOAN aditza (Verbo IR)

Ni parkera ______. (Yo voy al parque.)

noa
zoaz
doa
nator

Bikain! "Ni" + "noa" = Yo voy

Ez zuzena. Con "Ni" (yo), el verbo JOAN es "noa".

Zu nora ______? (Tu adonde vas?)

noa
zoaz
doa
zatoz

Primeran! "Zu" + "zoaz" = Tu vas

Ez zuzena. Con "Zu" (tu), el verbo JOAN es "zoaz".

Haiek geltokira ______. (Ellos van a la estacion.)

doa
goaz
doaz
datoz

Oso ondo! "Haiek" + "doaz" = Ellos van

Ez zuzena. Con "Haiek" (ellos), el verbo JOAN es "doaz".

3. Ariketa: ETORRI aditza (Verbo VENIR)

Ni Madriletik ______. (Yo vengo de Madrid.)

noa
nator
dator
gatoz

Bikain! "Ni" + "nator" = Yo vengo

Ez zuzena. Con "Ni" (yo), el verbo ETORRI es "nator".

Bera etxetik ______. (El/Ella viene de casa.)

nator
zatoz
dator
doa

Primeran! "Bera" + "dator" = El/Ella viene

Ez zuzena. Con "Bera" (el/ella), el verbo ETORRI es "dator".

Gu landetik ______. (Nosotros venimos del trabajo.)

goaz
gatoz
datoz
zatozte

Oso ondo! "Gu" + "gatoz" = Nosotros venimos

Ez zuzena. Con "Gu" (nosotros), el verbo ETORRI es "gatoz".

4. Ariketa: Norabideak (Direcciones)

Como se dice "a la izquierda" en euskera?

ezkerrera
eskuinera
zuzen
aurrera

Bikain! "Ezkerrera" = a la izquierda

Ez zuzena. Izquierda = ezkerrera, derecha = eskuinera.

Que significa "hurbil dago"?

Esta lejos
Esta cerca
Esta detras
Esta delante

Oso ondo! "Hurbil dago" = Esta cerca

Ez zuzena. Hurbil = cerca, urrun = lejos.

5. Ariketa: Hiriko lekuak (Lugares)

Como se dice "la estacion" en euskera?

aireportua
geltokia
ospitalea
farmazia

Bikain! "Geltokia" = la estacion

Ez zuzena. Estacion = geltokia, aeropuerto = aireportua.

Cual es el nombre vasco de Bilbao?

Donostia
Bilbo
Gasteiz
Irunea

Primeran! Bilbao = Bilbo en euskera

Ez zuzena. Bilbao = Bilbo, San Sebastian = Donostia, Vitoria = Gasteiz.

Como preguntas "Donde esta la farmacia?"

Nora dago farmazia?
Non dago farmazia?
Nondik dago farmazia?
Nola dago farmazia?

Oso ondo! "Non dago...?" = Donde esta...?

Ez zuzena. Para preguntar ubicacion usamos "Non dago...?"

PRAKTIKA GEHIAGO

Ariketa hauek egin ondoren, saiatu txatean tutorearekin:

  • Galdetu norabideak: "Non dago geltokia?"
  • Esan nora zoazen: "Bilbora noa"
  • Esan nondik zatozen: "Madriletik nator"

Despues de estos ejercicios, intenta en el chat con el tutor:

  • Pregunta direcciones: "Donde esta la estacion?"
  • Di adonde vas: "Voy a Bilbao"
  • Di de donde vienes: "Vengo de Madrid"
