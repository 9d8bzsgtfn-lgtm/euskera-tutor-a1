Praktika: Birpasa Orokorra (Repaso General)

ARGIBIDEAK

Este es un repaso completo de todo el curso A1. Incluye ejercicios de casos, verbos, vocabulario y estructuras.

Hau A1 kurtso osoaren birpasa da. Kasuak, aditzak, hiztegia eta egiturak sartzen dira.

1. Ariketa: Kasuak (Casos gramaticales)

Ume______ etorri da. (El nino ha venido.) - Caso NOR

-a
-ak
-ari
-an

Oso ondo! Caso NOR singular: -a (umea)

Ez zuzena. El caso NOR para sujeto intransitivo singular es -a.

Gizon______ liburua irakurri du. (El hombre ha leido el libro.) - Caso NORK

-a
-ak
-ari
-an

Bikain! Caso NORK singular: -ak (gizonak)

Ez zuzena. El caso NORK para sujeto transitivo singular es -ak.

Am______ oparia eman diot. (He dado el regalo a mama.) - Caso NORI

-a
-ak
-ari
-an

Primeran! Caso NORI: -ari (amari)

Ez zuzena. El caso NORI para objeto indirecto es -ari.

Etxe______ nago. (Estoy en casa.) - Caso NON

-a
-ra
-an
-tik

Oso ondo! Caso NON (ubicacion): -an (etxean)

Ez zuzena. El caso NON para ubicacion estatica es -an.

Bilbo______ noa. (Voy a Bilbao.) - Caso NORA

-n
-ra
-tik
-ko

Bikain! Caso NORA (direccion): -ra (Bilbora)

Ez zuzena. El caso NORA para direccion es -ra.

2. Ariketa: Aditzak (Verbos - tiempos verbales)

Ni ikaslea ______. (Yo soy estudiante.) - Presente IZAN

naiz
nintzen
nago
dut

Oso ondo! IZAN presente con "ni" es "naiz".

Ez zuzena. "Ni" + IZAN presente = naiz.

Atzo Bilbon ______. (Ayer estaba en Bilbao.) - Pasado IZAN

naiz
nintzen
nago
nengoen

Bikain! IZAN pasado con "ni" es "nintzen".

Ez zuzena. "Ni" + IZAN pasado = nintzen.

Nik ogia ______. (Yo como pan.) - Presente habitual

jaten dut
jan dut
jango dut
jan nuen

Primeran! Presente habitual: -ten + auxiliar presente (jaten dut).

Ez zuzena. El presente habitual se forma con -ten + auxiliar: jaten dut.

Atzo liburua ______. (Ayer lei el libro.) - Pasado

irakurtzen dut
irakurriko dut
irakurri nuen
irakurri dut

Oso ondo! Pasado: participio + auxiliar pasado (irakurri nuen).

Ez zuzena. El pasado se forma con participio + auxiliar pasado: irakurri nuen.

Bihar lanera ______. (Manana ire al trabajo.) - Futuro

joaten naiz
joan nintzen
joango naiz
joan naiz

Bikain! Futuro: -ko/-go + auxiliar presente (joango naiz).

Ez zuzena. El futuro se forma con -ko/-go + auxiliar: joango naiz.

3. Ariketa: Hiztegia (Vocabulario)

"Eskerrik asko" zer esan nahi du? (Que significa "Eskerrik asko"?)

Por favor
Muchas gracias
Perdon
De nada

Oso ondo! Eskerrik asko = Muchas gracias.

Ez zuzena. "Eskerrik asko" significa "muchas gracias".

Nola esaten da "hermano" euskaraz?

arreba
anaia
semea
alaba

Bikain! Anaia = hermano.

Ez zuzena. "Hermano" es "anaia". "Arreba" es hermana.

"Euria ari du" zer esan nahi du?

Hace sol
Hace frio
Esta lloviendo
Hace calor

Primeran! Euria ari du = Esta lloviendo.

Ez zuzena. "Euria ari du" significa "esta lloviendo".

Non erosten da janaria? (Donde se compra comida?)

ospitalean
dendan / supermerkatan
geltokian
museoan

Oso ondo! Janaria dendan edo supermerkatuan erosten da.

Ez zuzena. La comida se compra en la tienda (denda) o supermercado.

Nola esaten da "manana" (el dia siguiente)?

atzo
gaur
bihar
etzi

Bikain! Bihar = manana.

Ez zuzena. "Manana" es "bihar". "Atzo" es ayer, "gaur" es hoy.

4. Ariketa: Egiturak (Estructuras)

Pozik nago lana ______. (Estoy contento porque tengo trabajo.) - Causal

duela
dudalako
dudan
dut

Oso ondo! -elako expresa causa (porque).

Ez zuzena. La subordinada causal usa "-elako": dudalako.

Badakit etorriko ______. (Se que vendra.) - Completiva

dela
delako
den
da

Bikain! -ela introduce subordinadas completivas (que).

Ez zuzena. La subordinada completiva usa "-ela": dela.

Euskara ikasi ______ dut. (Quiero aprender euskera.)

nahi
behar
ahal
ari

Primeran! "Nahi" expresa deseo (querer).

Ez zuzena. "Querer" se expresa con "nahi": ikasi nahi dut.

Lanera joan ______ dut. (Tengo que ir al trabajo.)

nahi
behar
ahal
ari

Oso ondo! "Behar" expresa necesidad/obligacion (tener que).

Ez zuzena. "Tener que" se expresa con "behar": joan behar dut.

Euskaraz hitz egiten ______ naiz. (Estoy hablando en euskera.)

nahi
behar
ahal
ari

Bikain! "-tzen ari" expresa accion en progreso (estar + gerundio).

Ez zuzena. El progresivo usa "-tzen ari": hitz egiten ari naiz.
